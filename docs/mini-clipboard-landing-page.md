# Mini Clipboard（macOS）产品落地页功能与设计文档

## 1. 品牌与定位
- 品牌主张：把临时剪贴板升级为可搜索、可组织、可协作的信息库。
- 目标用户：开发者、写作者、设计师、运营等高频复制粘贴人群。
- 差异化：三种布局随心切换、Pinboards 收藏组织、全键盘高效操作、隐私友好。
- 期待体验：呼出面板 ≤ 150ms、搜索响应 ≤ 200ms。

## 2. 转化目标
- 主要 CTA：下载与安装（macOS），次级 CTA：观看演示视频、查看使用指南。
- 关键指标：落地页转化率、视频播放率、截图滑动率、CTA 点击率、页面停留时长。

## 3. 信息架构（IA）
1) Hero 区（价值主张 + 立即下载）
2) 即刻提效（核心卖点卡片）
3) 关键功能（时间线、搜索、Pinboards、快速粘贴、批量操作、预览与轻编辑、隐私）
4) 快速演示（短视频/动图）
5) 高级能力（组合过滤、纯文本粘贴、跨设备同步）
6) 安全与隐私（仅本地）
7) 用户评价与社会证明（未来拓展）
8) 常见问题（FAQ）
9) 底部 CTA 与资源（下载、使用指南、快捷键说明）

## 4. 文案草案（中文）
- Hero 标题：把复制粘贴，提升到专业生产力水平
- Hero 副标题：无限历史、秒速搜索、顺序粘贴与收藏组织，让跨应用拼装更高效。
- Primary CTA：立即下载（macOS）
- Secondary CTA：观看 30 秒演示
- 卖点卡片：
  - 全局时间线：找回最近 100 次复制，按来源 App 分类与预览。
  - 极速搜索：输入即搜，类型/来源组合过滤，≤ 200ms 响应。
  - 三种布局随心切换：时间线、列表、网格。
  - 批量操作：批量选中、复制、删除、加入 Pinboard。
  - Pinboards 收藏：常用模板与片段一键投递，支持拖拽排序。
  - 快速/纯文本粘贴：数字快捷键或双击选中并复制。

## 5. 关键功能展示
- 时间线浏览与导航：`⇧+⌘+P` 呼出/隐藏，方向键左右移动，`Space` 预览。
- 搜索与过滤：关键词 + 类型（Text/Image/Link/File）+ 来源 App（Safari/Notes 等）。
- Pinboards：颜色标记、拖拽排序、常用片段固定与一键投递。
- 快速粘贴与纯文本：`⌘+1…9` 快速选中并复制。
- 预览与轻编辑：Quick Look、多类型缩略图、文本编辑与重命名。
- 隐私与规则：历史保留周期、忽略敏感应用、屏幕共享自动隐藏。

## 6. 视觉设计系统
- 颜色主题（浅/深色）：
  - 星云幻境（Hero 渐变）：`#7C3AED → #22D3EE`，暗夜版 `#1E293B → #0EA5E9`。
  - 青玉白瓷（内容背景）：浅色 `#F8FAFC`，深色 `#0B1220`。
  - 霓虹暗夜（强调与按钮）：主色 `#0EA5E9`，悬停 `#38BDF8`，压暗 `#0284C7`。
  - 语义色：成功 `#10B981`、警告 `#F59E0B`、危险 `#EF4444`。
- 字体与层级：
  - 标题：优先系统字体（SF Pro），H1 36–44、H2 28–32、H3 20–24，行高 1.2–1.3。
  - 正文：16–18，行高 1.6；次要说明 14。
  - 数字与快捷键提示使用等宽变体以增强可读性。
- 间距系统：8pt 网格；Hero 区上下 96–128；模块间 48；卡片内 16–24。
- 组件风格：圆角 12–16、阴影柔和（`shadow-md`）、边框 1px（`#E2E8F0`/暗色 `#334155`）。
- 暗色模式：默认跟随系统；确保对比度符合 WCAG AA（≥ 4.5:1）。

## 7. 组件与模块
- HeroSection：标题、副标题、CTA、键盘动效背景（轻微视差）。
- FeatureCard：图标/缩略图 + 标题 + 说明 + 次级 CTA。
- TimelineViewer：横向时间线卡片，支持多选与键盘导航。
- SearchBar：输入即搜，下拉类型/来源过滤，显示匹配高亮。
- PinboardTabs：标签页 + 列表，支持拖拽排序与右键菜单。
- StackBanner：栈激活提示与方向切换。
- KeyboardHints：常用快捷键可视化（`⇧+⌘+P`、`⌘+1…9` 等）。
- VideoBlock：演示视频（懒加载，静音自动播放，手动循环）。
- PrivacyBlock：隐私原则与开关说明（仅本地）。
- FAQAccordion：常见问题折叠列表。
- Footer：下载、指南、反馈与版权。

## 8. 交互与动效
- 首屏进入：Hero 渐变缓入（300ms），标题淡入与上移 8pt。
- 卡片悬停：提升阴影与轻微缩放（1.02x），按钮背景过渡 200ms。
- 键盘提示动效：循环闪烁展示 `⇧+⌘+V`，强调“全键盘高效”。
- 时间线预览：滚动出现时渐入，预览浮层淡入/淡出 150ms。
- 视频：进入视口后懒加载，播放按钮具备点击反馈与波纹。

## 9. 技术实现建议（Next.js + shadcn/ui + Tailwind + TS）
- 框架：Next.js 13+（App Router），SSR + 选择性客户端组件以优化首屏与 SEO。
- UI：shadcn/ui 作为组件基，TailwindCSS 快速样式并接入设计令牌。
- 资产：`next/image` 优化截图与封面；`<video>` H.264/MP4 懒加载。
- 可访问性：语义标签、`aria-*`、焦点状态、键盘导航与对比度合规。
- 国际化：中文首发，预留英文；Meta 与 Open Graph 遵循本地化。

## 10. SEO 与性能
- 结构化标题：单页 H1 唯一，H2–H3 分区；语义 HTML（`section`/`header`/`footer`）。
- 元数据：`title`/`description`/OG/Twitter Cards；`canonical`；站点图与 robots。
- Schema 标记：`SoftwareApplication` + `VideoObject`（如有演示视频）。
- 性能：CLS/LCP 友好布局、资源预加载、图片懒加载、字体回退与子集化。

## 12. FAQ（示例）
- 是否读取隐私内容？默认忽略敏感应用；可关闭同步，数据仅本地或 iCloud 私有库。
- 直接粘贴如何工作？需开启辅助功能权限，用于向前台应用发送 `⌘+V`。
- 多设备同步？同一 Apple 账户下，通过 iCloud 私有库增量同步。
- 是否支持纯文本粘贴？支持，`⇧` 修饰触发纯文本粘贴。

## 13. 埋点与 A/B 测试
- 事件：CTA 点击、视频播放/完成、截图滑动、FAQ 展开、Hero 到达率。
- 变量：Hero 标题（效率 vs 专业）、CTA 文案（“立即下载” vs “免费安装”）、卖点排序。
- 指标：CTR、播放率、滚动深度、转化率、退出率；按流量分桶对比。

## 14. 发布检查清单
- 内容：标题与卖点一致；截图与视频加载正常；FAQ 完整。
- 体验：暗/亮模式对比度合规；键盘/屏幕阅读器可用。
- 技术：元数据与 Schema 正确；资源懒加载；核心指标达标。
- 法务与隐私：用途说明清晰；权限引导合规；数据去向透明。
